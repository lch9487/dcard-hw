## Installation

```shell
$ git clone git@github.com:lch9487/dcard-hw.git
$ cd dcard-hw
$ yarn
$ yarn start
```

## Warning

If you encounter warnings in vscode after `yarn start`, please go to `tsconfig.json`
and change `"jsx": "react-jsx"` to `"jsx": "react"`
